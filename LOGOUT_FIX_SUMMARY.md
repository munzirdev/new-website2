# إصلاح مشكلة تسجيل الخروج

## المشكلة
عند تسجيل الخروج وتحديث الصفحة، كان المستخدم يعود كأنه مسجل دخول بسبب عدم تنظيف localStorage و sessionStorage بشكل صحيح.

## الإصلاحات المطبقة

### 1. تحسين دالة `signOut` في `useAuth.ts`
- إضافة تنظيف شامل لـ localStorage
- إضافة تنظيف شامل لـ sessionStorage
- إضافة تنظيف جلسة Supabase من localStorage
- إضافة علامة `manuallySignedOut` لتتبع تسجيل الخروج اليدوي

### 2. تحسين `onAuthStateChange` listener
- إضافة معالجة فورية لحدث `SIGNED_OUT`
- منع إعادة تحميل المستخدم عند تسجيل الخروج

### 3. تحسين `getInitialSession`
- إضافة فحص لعلامة `manuallySignedOut`
- تجاهل الجلسة المحفوظة إذا كان المستخدم قد سجل خروج يدوياً

### 4. تحسين `simpleSignOut`
- إضافة تنظيف شامل للـ storage
- إضافة علامة `manuallySignedOut`

### 5. تحسين تنظيف البيانات في `App.tsx`
- إضافة تنظيف شامل لجميع البيانات عند تسجيل الخروج
- تنظيف علامات الزيارة والإشعارات

## الملفات المعدلة
- `src/hooks/useAuth.ts`
- `src/App.tsx`

## كيفية الاختبار
1. سجل دخول كأي مستخدم
2. اضغط على زر تسجيل الخروج
3. حدث الصفحة (F5)
4. تأكد من أن المستخدم لم يعد مسجل دخول

## النتائج المتوقعة
- تنظيف كامل لجميع البيانات المحفوظة
- عدم إعادة تسجيل الدخول عند تحديث الصفحة
- تجربة تسجيل خروج سلسة ومتسقة
